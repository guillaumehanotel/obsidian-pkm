# Solutions d'authentification sur Laravel


## [Laravel Breeze](https://laravel.com/docs/9.x/starter-kits#laravel-breeze) ✔️
**TailwindCSS + Blade ou InertiaJS**
Connexion - Enregistrement
Réinitialisation MDP
Vérification Email

## [Laravel Jetstream](https://laravel.com/docs/9.x/starter-kits#laravel-jetstream) ✔️
**TailwindCSS +Livewire ou InertiaJS**
Connexion - Enregistrement
Acceptation des conditions générales
Rénitialisation & Confirmation de MDP
Vérification Email
Plus de fonctionnalités :
- Gestion de profil user
- 2FA
- API pour SPA
- Gestion des équipes

Complexe à personnaliser

## [Laravel Fortify](https://laravel.com/docs/9.x/fortify)
**Sans Frontend**
**C'est le code Back-end utilisé pour Jetstream Breeze**
Connexion - Enregistrement
Réinitialisation MDP
Vérification Email
2FA

## [Laravel Sanctum](https://laravel.com/docs/9.x/sanctum)
**API Backend**
Auth pour SPA
Basé sur les Tokens avec scope & responsabilités
Si besoin 0auth : Laravel Passport

## [Laravel UI](https://github.com/laravel/ui)
**Package "abandonné" mais toujours maintenu
Bootstrap ou Vue ou React** 
Connexion - Enregistrement
Réinitialisation MDP
Vérification Email

## [Laravel Spark](https://spark.laravel.com/) (payant)
Portail prêt à l'emploi
Inclus toutes les fonctionnalités nécessaires à un Saas pour vendre des applications via Paypal, Stripe, ...
